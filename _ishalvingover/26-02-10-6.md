<h3>The Subdraft.</h3>

<p class="text-align-center">Config[t] = { Substance<sub>0</sub>[t] Substance<sub>1</sub>[t] ... Substance<sub>N-1</sub>[t] }</p>

A configuration is a set of subtances, whose number is many yet finite.

Each substance has its own state, which is low-informational. 

Substances update their state by time; the nonlinear function `f` below is stochastic and emits probablities for next possible states. The symbol `<-` denotes that one of the state is selected according to the probablities.

<p class="text-align-center">Substance<sub>i</sub>[t+1] <- f(Neighbors(Substance<sub>i</sub>[t]))</p>

where

<p class="text-align-center">Neighbors(Substance<sub>i</sub>[t])</p>
<p class="text-align-center">= {Substance<sub>j</sub>[t]: | Substance<sub>i</sub>[t].position - Substance<sub>j</sub>[t].position | <= 1}</p>

That is, the `Neighbors` of a substance is a set of substances that are spatially immediate, & the substance itself.

- The number of the substances does not change by time; the boundary substances are padded. The decrease in the configuration's size is of the probability.

Since this update is synchronized, a configuration can be expressed as an output of the global transition function `F`:

<p class="text-align-center">Config[t+1]</p>
<p class="text-align-center"><- F(Config[t])</p>
<p class="text-align-center">= { f(Neighbors(Substance<sub>0</sub>[t])) ... f(Neighbors(Substance<sub>N-1</sub>[t])) }</p>

The function `F` (and `f` [error; to be rewritten]) can be decomposed with respect to time, until the atomic time-unit.

<p class="text-align-center">F(·) = F<sup>1</sup>(·) = F<sup>1/2</sup>(F<sup>1/2</sup>(·)) = ...</p>
