<h3>The Subdraft.</h3>

<p class="text-align-center">Config[t] = { Substance<sub>0</sub>[t] Substance<sub>1</sub>[t] ... Substance<sub>N-1</sub>[t] }</p>

A configuration is a set of subtances, whose number is many yet finite.

Each substance has its own state, which is low-informational. 

Substances update their state by time; the nonlinear function `f` below is stochastic and emits probablities for next possible states. The symbol `<-` denotes that one of the state is selected according to the probablities.

<p class="text-align-center">Substance<sub>i</sub>[t+1] <- f(Neighbors(Substance<sub>i</sub>[t]))</p>

where

<p class="text-align-center">Neighbors(Substance<sub>i</sub>[t])</p>
<p class="text-align-center">= {Substance<sub>j</sub>[t]: | Substance<sub>i</sub>[t].position - Substance<sub>j</sub>[t].position | <= 1}</p>

That is, the `Neighbors` of a substance is a set of substances that are spatially immediate, & the substance itself.

The number of the substances does not change by time; the boundary substances are padded. The decrease in the configuration's size is of the probability.

Since this update is synchronized, a configuration can be expressed as an output of the global transition function `F`:

<p class="text-align-center">Config[t+1]</p>
<p class="text-align-center"><- F(Config[t])</p>
<p class="text-align-center">= { f(Neighbors(Substance<sub>0</sub>[t])) ... f(Neighbors(Substance<sub>N-1</sub>[t])) }</p>

The atomic time-unit is quite short; for longer intervals the `F` can be simply composed:

<p class="text-align-center">Config[t+1ms] <- F<sup>MS</sup>(Config[t])</p>
<p class="text-align-center">where F<sup>MS</sup>(·) = F(F(...(F(·))))</p>
<p class="text-align-center">where MS * 1 (atomic time-unit) = 1ms</p>

If one observe a trajectory, he will find that substances can be categorized into two regions `Imaginary` & `Real`, distinguished by some portion of the state, which is maintained through time. This implies that the transition function leaves it almost surely intact. One now wonders...

1. The macroscopic organization, composed of substances that are updated by the microscopic transition function, since arises only by the said microscopic transition function, has no end other than it is spontaneous result of it; the probabilities of the successing configurations (conditioned on the earlier configuration) are merely the joint probabilities of the miscroscopic probabilities.

2. Yet the transition function can be any; and hence there can be any trajectories, with all possible sequence of configurations and the probabilities per configuration.

3. One sees latent divinities -- where sit they? 
  - Certainly not on the material-plane: if it was the case, since all substances are impure and contain nonzero probabilities for all region in the material-plane; hence ex hypothesi all substances would contain some amount of divinity, which is false.
  - One could argue that they are made into the trajectories. But if that was the case, one could simply behold the "divine" portion of the trajectory to see the divinities. But this cannot be.
  - It follows the divinities shall lie in the motion that is the transition function.

4. It is sure that not all outcomes that the transition function produces are divine. What is too be examined, tho', is whether the probabilities of the configuration have significance, especially on whether the trajectory follows the supposed nature.

5. This presumes an end of the trajectory -- yet the configuration only diminishes by size asymptotically, since the substances are impure and there cannot be a transition function that perfectly surely puts the probability `1` for a single configuration.
